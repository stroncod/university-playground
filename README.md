Proyecto Arquitecturas Computacionales UNAB 2016

Bloque que se mueve por 3 sensores marcando cada posición utilizando la tecla espacio.


1.Primer sensor
2.Primer y segundo sensor
3.Segundo sensor
4.Segundo y tercer sensor
